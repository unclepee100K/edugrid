
What is EduGrid?
EduGrid is a complete web-based School Management System designed specifically for Zimbabwean secondary schools (O-Level and A-Level). It connects parents, teachers, students, and administrators in one secure digital platform.

How It Works – In Simple Terms
🏫 For the School (Admin)
Parents apply online → Admin reviews applications → Accept/Decline/Waitlist.

When accepted, the system automatically creates a student profile, generates a unique Student ID (e.g., GRD-2026-SCI-001), and creates a login account.

Admin manages subjects, terms, fee structures, and publishes news/events.

All data is centralised – no more paper files!

👩‍🏫 For Teachers
Log in to a teacher dashboard.

Create assignments (set subject, due date, max marks, upload documents).

View pending student submissions, mark them, and add feedback.

Marks update instantly on the student's dashboard.

🧑‍🎓 For Students
Log in to a personal dashboard.

See overall average, pending assignments, and recent submissions.

View interactive charts showing subject performance and termly progress.

Submit assignments (upload file or type response).

Download official ZIMSEC-style PDF report cards at the end of each term.

👪 For Parents
Apply for their child online (no login required).

Receive application reference number and status updates via SMS/email.

After acceptance, log in using Student ID + 4-digit PIN.

View child's progress, marks, reports, and fee balance – read-only access.

💡 Why EduGrid?
Paperless – everything is digital.

Real-time – parents and students see progress instantly.

ZIMSEC-compliant – reports follow official grading.

Zimbabwe-focused – supports dual currency (USD/ZWG), low-data mode, and local phone numbers.

All-in-one – from applications to alumni.
